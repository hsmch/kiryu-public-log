# CLAUDE.md - Kiryu Public Log プロジェクトコンテキスト

## プロジェクト概要

- **サービス名**: Kiryu Public Log（略称: KPL）
- **URL**: kiryu.co
- **リポジトリ**: hsmch/kiryu-public-log
- **運営**: 細道（hsmch） - 個人事業主の事業の一つとして運営
- **目的**: 桐生市の市政・議会の公開情報を自動収集・構造化し、市民にわかりやすく公開する中立的なアーカイブサイト

## 設計方針

- 更新はなるべく自動化し、人手をかけない運用を目指す
- AI要約には「AI要約」であることを明示し、原文リンクを必ず併記する
- 中立的なアーカイブとして、情報の加工は最小限にとどめる
- 公式情報への入り口となる位置づけ（サイト内で完結させない）

## 技術スタック（予定）

- **サイト生成**: Astro（静的サイト生成）
- **ホスティング**: Cloudflare Pages
- **データ収集**: スクレイパー（定期実行: GitHub Actions）
- **データ解析**: Claude API（PDF構造化、要約、タグ付け）
- **データ保存**: JSON（data/ ディレクトリ）
- **ドメイン**: kiryu.co（メイン）、kiryu.life（将来の暮らし情報向け）

## リポジトリ構成

```
hsmch/kiryu-public-log/
├── site/          # Astro 静的サイト本体
├── pipeline/      # スクレイパー・データ収集パイプライン
├── data/          # 構造化された JSON データ
├── scripts/       # ユーティリティスクリプト
├── CLAUDE.md      # このファイル
└── README.md
```

## データソースと収集戦略

### 桐生市公式サイト (city.kiryu.lg.jp)

#### Phase 1（優先）

1. **議員名簿** (`/shigikai/about/1003765.html`)
   - 議席番号・氏名・会派・常任委員会・当選回数
   - HTMLが構造的でパースしやすい
   - 更新頻度: 選挙後（4年に1回）、役職改選時

2. **議案・採決結果** (`/shigikai/honkaigi/gian/`)
   - 定例会（年4回: 3月・6月・9月・12月）ごとにページが作成される
   - 「議案請願議決結果」のページから構造化

3. **新着情報** (`/shigikai/index.html`)
   - トップページの「新着更新情報」セクションの差分検知
   - 本会議日程、一般質問通告一覧なども含む

#### Phase 2

4. **会議録**
   - 会議録検索システム (ssp.kaigiroku.net) は robots.txt でクロール拒否
   - 代替: 市議会だよりPDFからの情報抽出、YouTube議会中継の文字起こし

5. **予算・財政ダッシュボード**
   - 予算書・決算書はPDF配布が中心
   - 群馬県サイトから Excel 形式の財政状況資料集も取得可能
   - 年1-2回更新なので半手動でも可

#### Phase 3

6. **テーマ別横断整理**
   - Claude APIで議案・一般質問・委員会結果を横断的にタグ付け・関連付け

### URL構造の規則性

桐生市サイトのURLは数字IDベースで規則的:
- 日程: `/shigikai/honkaigi/nittei/{年度ID}/{定例会ID}.html`
- 議案: `/shigikai/honkaigi/gian/shichogian/{年度ID}/...`
- 委員会: `/shigikai/1012710/1012992/...`
- 予算: `/shisei/zaisei/1006758/{年度ID}/...`

### 議会の基本情報

- 定例会: 年4回（3月・6月・9月・12月）
- 議員定数: 22名
- 会派: 一心会、そうぞう未来、政策研究会、公明クラブ、クラブ21、日本共産党議員団、無会派
- 常任委員会: 総務委員会、教育民生委員会、経済建設委員会
- 議会SNS: Facebook、X、LINE、YouTube

## サイト構成（予定）

- **トップ**: 最新の動きをタイムライン表示
- **議会**: 定例会ごとの議案一覧・採決結果・一般質問
- **議員**: 議員別のプロフィール・活動記録
- **まちの数字**: 人口・財政・予算のダッシュボード
- **テーマ別**: 継続的なトピックの経緯まとめ
- **about**: サイトの趣旨・中立性の方針・運営者情報

## 運用コスト見積もり

- ホスティング（Cloudflare Pages）: 無料
- ドメイン: 年数千円
- Claude API（要約・解析）: 月数百〜数千円
- GitHub Actions（定期実行）: 無料枠内
